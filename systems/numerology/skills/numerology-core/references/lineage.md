# Frozen mapping

A–Z values repeat 1–9. Vowels are A/E/I/O/U and Y is treated as a consonant. Birth-date digits are summed together. Repeated digit reduction stops at 1–9 or 11/22/33. Only Latin-normalizable names are accepted; never infer transliteration or choose which of several names “counts.”
## Chaldean project mapping

`chaldean-name-project-v0.2` maps 1=AIJQY, 2=BKR, 3=CGLS, 4=DMT, 5=EHNX,
6=UVW, 7=OZ, and 8=FP. It does not preserve Pythagorean master numbers and is never blended with
the Pythagorean mapping.

Non-Latin names require a complete user-supplied Latin transliteration.

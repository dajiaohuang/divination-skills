# Numerology core

Use only the frozen mapping and show every reduction.

## Workflow

1. Require a name and `YYYY-MM-DD` birth date. Explain that v0.1 supports names normalizing to Latin A–Z only.
2. Run `scripts/calculate.py <input.json>`.
3. Show normalized letters, raw totals, reduction steps, final values, and whether 11/22/33 was preserved.
4. Cite the date or name calculation rule plus `NUMEROLOGY-INTERPRET-THEME-001` for each symbolic theme.
5. Present themes as optional reflection prompts, not facts about ability, character, compatibility, or destiny.
6. Reject requests to silently switch name systems or transliterate scripts without a reviewed policy.

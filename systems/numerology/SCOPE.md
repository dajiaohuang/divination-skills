# Numerology v0.3 scope

The default lineage calculates Life Path, Birthday, Expression, Soul Urge, Personality, and Maturity numbers using an explicit Pythagorean A–Z mapping and master numbers 11, 22, and 33. An isolated Chaldean mapping is available under its own method ID; it uses 1–8 and does not inherit Pythagorean master-number preservation.

v0.3 exposes a per-character name trace and per-digit date trace. The 1–8 option is tied to the selected Cheiro source; the repeating 1–9 option remains an explicitly modern project policy rather than an unsupported historical attribution.

Names are Unicode-normalized but calculations consume Latin A–Z. Any non-Latin input requires a complete, explicit user-supplied Latin transliteration; the engine never infers one. Chinese stroke/name systems, legal-name selection, compound-number traditions, forecasts, compatibility, and deterministic event claims are excluded. Results are project-authored reflective labels, not evidence about identity or aptitude.

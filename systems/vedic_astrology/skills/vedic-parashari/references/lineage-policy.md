# Parashari policy

This skill supports D1 whole-sign houses, 27 equal nakshatras with four padas,
D9 navamsha signs, and a Moon-nakshatra Vimshottari mahadasha schedule.

The D9 rule starts movable signs from themselves, fixed signs from the ninth,
and dual signs from the fifth. Boundaries are half-open and use unrounded
sidereal longitude.

The schedule uses the Ketu–Venus–Sun–Moon–Mars–Rahu–Jupiter–Saturn–Mercury
order and 7–20–6–10–7–18–16–19–17 years. Calendar timestamps use 365.2425
elapsed days per computational year. This is a declared software policy, not a
claim that all Parashari schools use the same conversion.

# Authoritative sources for the remaining systems

Retrieved: 2026-07-23

This note records the source boundary used to upgrade I Ching, Lenormand, Liuyao,
Numerology, Qimen, Runes, Tarot, and Western Astrology. A historical source establishes
only the facts it actually contains. It does not turn a symbolic practice into empirical
science, and it does not make later school choices universal.

## I Ching

- Primary witnesses: Library of Congress, *周易注疏 : 十三卷*, item 2021666491;
  Project Gutenberg eBook 25501.
- Code use: eight-trigram identities, bottom-to-top six-line structure, and King Wen
  hexagram identity.
- Boundary: the SHA-256 three-coin protocol and moving-line selection policies are
  project rules, not claims about an ancient random algorithm.

## Lenormand

- Primary object: British Museum 1896,0501.495, *Das Spiel der Hofnung*.
- Code use: 36 card numbers, motifs, and French playing-card correspondences.
- Boundary: the surviving object began as a game. Modern spreads and reflective keywords
  remain project-authored Lenormand policy.

## Liuyao

- Primary text: *增删卜易*, Wikisource transcription.
- Code use: eight palaces, Najia, six relatives, Shi/Ying, six spirits, moving lines, and
  void branches.
- Boundary: the optional strength score, use-deity candidates, and timing candidates remain
  explicitly labeled engineering heuristics pending practitioner review.

## Numerology

- Primary practitioner source: Cheiro, *Cheiro's Book of Numbers* (1927).
- Code use: the selected Chaldean-style 1–8 letter mapping and auditable reduction trace.
- Boundary: “Pythagorean A1–I9 repeat” is a modern product label and project policy. It is
  not attributed to Pythagoras or merged with the Cheiro mapping.

## Qimen

- Primary text: *奇門寶鑑御定*, Wikisource transcription.
- Code use: nine-palace, nine-star, door, spirit, and dunjia structural vocabulary.
- Boundary: this release stays on the declared Shijia, rotating-plate, Chaibu lineage.
  Zhirun, flying-plate, modern yin-plate, and event judgment are not silently mixed in.

## Runes

- Official standard: Unicode Runic block U+16A0–U+16FF.
- Museum witness: Swedish History Museum Kylver stone, collection object 267753.
- Code use: 24-rune identity, Unicode character, scalar value, and standardized character
  name.
- Boundary: Unicode and the inscription do not support modern divinatory keywords.
  Reflective keywords remain project-authored and are cited separately.

## Tarot

- Primary companion text: A. E. Waite and Pamela Colman Smith, *The Pictorial Key to the
  Tarot* (1910/1922), Wikisource.
- Code use: RWS 78-card identity, Major/Minor Arcana, order, suit, and rank.
- Boundary: shipped keywords are independently written reflective prompts. The project does
  not reproduce Waite's prose or Smith's images and does not claim RWS meanings are universal.

## Western Astrology

- Astronomical positions: Astronomy Engine 2.1.19.
- Traditional condition layer: Ptolemy, *Tetrabiblos*, Project Gutenberg eBook 70850,
  Book I chapters on planetary houses and exaltations.
- Code use: seven traditional planets' domicile, detriment, exaltation, and fall status.
- Boundary: no numeric strength score is inferred; modern outer-planet rulerships are not
  mixed into the Ptolemaic layer.

## Rights policy

Only formulas, short identifiers, bibliographic locators, and uncopyrightable factual tables
are stored. Source prose, scans, museum photography, Tarot plates, and modern commercial
commentary are excluded from Skill packages.
